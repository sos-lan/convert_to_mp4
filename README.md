# convert_to_mp4
Бат-скрипт для потоковой конвертации из контейнеров avi и mkv в mp4 с преобразованием аудио-дорожки в aac stereo для Windows.
Скачать ffmpeg можно здесь: https://www.gyan.dev/ffmpeg/builds/ffmpeg-git-full.7z
Распаковать необходимо в эту папку: C:\ffmpeg\bin или изменить путь в скрипте.
